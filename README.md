# My Personal Diary 📒

**My Personal Diary** es una aplicación de escritorio minimalista y elegante diseñada para capturar pensamientos y vivencias diarias de forma organizada. El proyecto se enfoca en la facilidad de uso, permitiendo guardar registros automáticamente con la fecha actual.

## ✨ Funcionalidades Principales
* **Escritura Libre:** Área de texto amplia con soporte para múltiples líneas que facilita una redacción fluida.
* **Guardado Inteligente:** Las entradas se almacenan automáticamente con el formato `Diary_YYYY-MM-DD.txt` para mantener un orden cronológico perfecto.
* **Explorador de Archivos:** Capacidad para abrir y editar registros creados anteriormente de manera directa desde la interfaz.
* **Interfaz Personalizada:** Diseño visual limpio que utiliza tipografía Georgia y Calibri con colores suaves para mejorar la experiencia de escritura.

## 🛠 Tecnologías Utilizadas
* **Python:** Lenguaje de programación principal para el desarrollo de la lógica.
* **Tkinter:** Librería estándar de Python utilizada para la creación de la interfaz gráfica de usuario (GUI).
* **File System (OS):** Manejo de persistencia de datos mediante archivos de texto plano (.txt).

## 🚀 Instalación y Ejecución

Para ejecutar este proyecto en tu entorno local, asegúrate de tener instalado Python en tu sistema y sigue estos pasos:


# 1. Clona este repositorio
```bash
git clone https://github.com/ClutchDev23/Simulador-de-diario-personal.git
```
# 2. Accede a la carpeta del proyecto
```bash
cd "Simulador de Diario personal"
```
# 3. Inicia la aplicación
```bash
python Diario.py
````
